## Image Specifications

- **Format:** JPG or PNG
- **Recommended size:** 800-1200px width
- **Aspect ratio:** 4:3 or 16:9 preferred

## Fallback Behavior

If images are not present, the page will automatically show placeholder images with the hotel name. The Google Maps embeds will still display the hotel locations with street view and photos from Google.

## Tips for Finding Images

1. **Hotel Official Websites** - Best quality, properly licensed
2. **Google Images** - Use with proper attribution
3. **Wikimedia Commons** - Free-to-use images
4. **Your own photos** - If you've stayed at these hotels

**Note:** Ensure you have the right to use any images you download. Official hotel websites often allow use for personal, non-commercial purposes.
